## 安装

**注意**：确保Flutter开发环境已经搭建完成。记得打开一个设备。

在命令终端中，运行以下命令。

    git clone https://github.com/gyt95/flutter_demo.git
    cd flutter_demo
    flutter run -v -d emulator-5554

或者，如果您使用 Visual Studio Code，则可以打开 lib/main.dart 并按 F5 运行该应用程序。
